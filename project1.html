How it All Fits Together

The internet is a large amount of computers connected via a series of tubes that use these tubes to talk to each other. Ok the internet isn't a series of tubes, but it is a vast quantity of computers networked together. Some of these computers are standard personal computers, while some of them are servers. These servers contain vast amounts of data that personal computers can request access to via HTTP requests. The server then sends the requested information back to the personal computer which then displays it in a web browser.

Inline and Block Elements

Elements in HTML documents are either inline or block. Inline elements appear on the same line on which they were created. Whereas block elements draw a box around the elements inside of them.

How Websites Work

Websites display content by utilizing programming languages . The two languages that this section focuses on are CSS and HTML. CSS files are the style of the site, while HTML files control the structure of the site. If there are any interactive parts on a site they are usually handles with javascript. Programmers also utilize the DOM (document object model) to structure pages.

Everything is a Box

What's in the box? Well everything is in the box, or more correctly a box. Everything on a website is in a box, and within that box are boxes. Turns out it's boxes all the way down. When a building a web page one must think of how to boxify their designs. One defines boxes with the div tag and applies styles to them and gives them attributes.

HTML

HTML is the language of the web and as such it has rules and syntax. A basic "word" in HTML is known as a tag a web browser turns tags into elements using the DOM tree. Everything between the starting tag and the ending tag is referred to as content . All tags have a type as well as attributes and values.

CSS

CSS stands for cascading style sheets. The word cascading in this context refers to the fact that the most specific rule gets applied. For each html document you can reference several css files. In css you first write a selector which determines what that css elemnt gets applied to.

Style

In programming they word style can refer to many different things. Such as

  • An html element
  • An html attribute
  • The general look of a webpage
  • A sick basketball shot
  • Or as a verb, describing the process of applying style to a webpage
Important css terms
  • Rule: A line of css code describing the value a certain attribute should take.
  • Property: The property you want to change.
  • Value: The value you want to assign to an attribute.
  • Selector: The name that you use to target the elements that are assigned to a clas or id attribute in the html.
  • Class: Refers to a group of elements that can be selected together. Class names should not contain periods or puctuation!
  • ID:Id's are unique identifiers that identify an element in html.
