How it all fits together

The internet is a large amount of computers connected via a series of tubes that use these tubes to talk to each other. OK the internet isn't a series of tubes, but it is a vast quantity of computers networked together. Some of these computers are standard personal computers, while some of them are servers . These servers contain vast amounts of data that personal computers can request access to via HTTP requests . The server then sends the requested information back to the personal computer which displays it in a web browser .

Inline and Block Elements

Elements in HTML documents are either inline or block . Inline Elements appear on the same line upon which they were created. Whereas block elements have a box drawn around the elemnts inside them.

How Websites Work

Websites display content by utilizing programming languages . The two languages that this section focuses on are CSS and HTML . CSS files are the style of the site, while HTML files control the structure of the site. If there any interactive parts of the site they are usually handled with Javascript . Programers also utilizes the DOM (document object model) to structure pages.

HTML

HTML is the language of the web and as such it has rules and syntax. A basic "word" in HTML is known as a tag a web browser turns tags into elements Using the DOM tree. Everything between the starting tag and the ending tag is reffered to as content . All tags have a type as well as attributes and values.

Everything Is A Box

Whats in the box? Well everything is in the box, or more correctly A box. Every thing on a website is in a box, and within that box are more boxes. Turns out it's boxes all the way down. When building a web page one must think of how to boxify their designs. One defines boxes with the div tag and applies styles to them and gives them attributes.
